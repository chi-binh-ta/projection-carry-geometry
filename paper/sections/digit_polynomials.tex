\section{Digit Polynomials and Digit Measures}

Let $B \ge 2$ be a base. A nonnegative integer $X$ is written as
\[
  X = \sum_i a_i B^i,
\]
where each digit satisfies $0 \le a_i < B$. Throughout this project the digit
vector is ordered from low to high degree, so $123$ in base $10$ is represented
as $(3,2,1)$.

The digit support is
\[
  \operatorname{supp}(X) = \{i : a_i \ne 0\}.
\]
This support view connects multiplication to additive structure on degree
indices.
